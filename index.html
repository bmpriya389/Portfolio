Pension Simulation Calculator

Pension Simulation Calculator is an open source Calculator and Simulator built for the Rockefeller College of Public Affairs and Policy using R programming language.

The calculator examines the effects of change in actuarial inputs like discount rates, salary growth rate, amortization period etc. on the size of the public pension liabilities and contribution costs.

Client: David Matkin - Vice Chair of the Department of Public Administration and Policy, Assistant Professor
Gang Chen - Assistant Professor, Department of Public Administration and Policy

Date: August 2015

Service: Data Analytics and Web Development

Languages & packages: R, Shiny, AWS EC2 instance

Discovering Social Circles

Generate an automated solution for detecting social circles in social networking sites like Facebook, google plus etc. A dummy dataset of Facebook with different user profiles was utilized and network structure and profile information was used to base circle detection in a network.

Tasks

  • Generate an automated solution for detecting social circles in Facebook using k way partitioning while utilizing feature vector between each alter-ego and alter-alter pair to produce edge weights for such pairs
  • Form alter-alter and ego-alter circles and generated a final matrix using the two to determine social circles in an ego network

Professor: Petko Bogdanov - Assistant Professor

Date: August 2014

Service: Network Mining

Languages & packages: Python, C++, Matlab 2013, Gephi
